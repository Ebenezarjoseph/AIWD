
# Lab1
The portal allows users to view details of an upcoming event through HTML5 media elements (image banner, promotional video, background audio).  Users can then register for the event using an HTML5 form with various input types (text, email, date, radio buttons, select menus, textarea, file upload).

# Lab2
In this experiment, the HTML5-based Online Event Registration &amp; Promotional Portal created in Experiment 1 is enhanced using CSS and Advanced CSS3 techniques. Styling is applied to improve typography, colors, spacing, and layout.

# Lab3
The responsive website is developed using the Bootstrap framework to ensure consistency in
layout and design across various devices such as desktops, tablets, and mobile phones.
Bootstrap’s grid system, predefined CSS classes, and UI components are used to create a
flexible and responsive layout without writing extensive custom CSS.
This experiment demonstrates the effective use of Bootstrap to build modern, mobile-first, and
responsive web pages.
